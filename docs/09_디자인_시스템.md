## 1. 디자인 방향

### 1.1 키워드

**Clean · Trust · Efficient**

- **Clean**: 세탁소 = 깨끗함. 여백과 정돈된 그리드.
- **Trust**: 금액·상태 데이터는 명확하고 오해 없게.
- **Efficient**: 하루 종일 쓰는 데스크� — 눈 fatigue 최소, 키보드 친화.

### 1.2 레거시 대비 개선(의도적 변경)

| 레거시(DevExpress) | 차세대 |
|-------------------|--------|
| Windows XP풍 그레이 톤 | 밝은 Neutral + Teal 포인트 |
| 그리드 중심 단일 화면 | 사이드바 + 필터 + 마스터-디테일 |
| 상태 = 텍스트 | **색상 뱃지** + 아이콘 |
| 작은 폰트·조밀 | **14px 기본**, 행 높이 44px |

### 1.3 비목표

- Material 3 기본 테마 그대로 사용(커스텀 ThemeExtension 필수)
- 모바일 퍼스트 UI(터치 타겟 48px 강제 등)

## 3. Components

### 3.1 App Shell

```
┌─ Sidebar ─┬─ TopBar ─────────────────────────────┐
│  NavItem  │  Breadcrumb / Title    UserMenu      │
│  ...      ├─ FilterBar (optional) ───────────────┤
│           │  Content                              │
└───────────┴───────────────────────────────────────┘
```

- **Sidebar**: `brand.secondary` 배경, 활성 = `brand.primary` 좌측 3px bar + `neutral.0` 텍스트
- **TopBar**: `neutral.0` 배경, 하단 `neutral.200` border

### 3.2 Buttons

| Variant | 배경 | 텍스트 | 용도 |
|---------|------|--------|------|
| Primary | `brand.primary` | white | [로그인], [출고 처리] |
| Secondary | white | `neutral.900` | [취소], [닫기] |
| Ghost | transparent | `brand.primary` | [새로고침] |
| Danger | `semantic.error` | white | [접수 취소] |

- Height: 36px (compact), 40px (default)
- Min width: 80px
- Disabled: opacity 0.45

### 3.3 Input

- Height: 40px, border `neutral.200`, focus ring `brand.primary` 2px
- Label 위 배치, `body.sm` + `neutral.600`
- Error: border `semantic.error` + helper text

### 3.4 Data Table

- Header: `neutral.50` bg, `label` uppercase, sticky
- Row hover: `brand.primaryMuted` 50% opacity
- Row selected: `brand.primaryMuted` full
- Zebra: 사용 안 함(상태색과 충돌 방지)
- Empty: 일러스트 + "조건에 맞는 접수가 없습니다" + [필터 초기화]

### 3.5 Status Badge

```
┌──────────┐
│ ● 작업중  │  padding: 4px 10px, radius.sm, label
└──────────┘
```

- 좌측 dot 6px (텍스트색과 동일)
- 테이블 내 고정 width 금지 — 텍스트 길이 따름

### 3.6 Summary Bar (합계)

접수 목록 상단 — **병행 대조 핵심 UI**.

```
┌─────────────────────────────────────────────────────────┐
│  총 24건  │  ₩ 386,000  │  접수대기 3  작업중 8  완료 13  │
└─────────────────────────────────────────────────────────┘
```

- `title.lg` for 건수·총액, `body.sm` for 상태별
- 구분: `neutral.200` vertical divider

### 3.7 Filter Chips (기간 프리셋)

- Selected: `brand.primaryMuted` bg + `brand.primary` text
- Default: white + `neutral.200` border

### 3.8 Dialog / Confirm

- 상태 변경 등 destructive action: Primary 대신 **Secondary 확인 + Primary 실행**
- 본문에 현재→변경 상태 뱃지 나란히 표시

### 3.9 Toast / Snackbar

- Success: 3초 auto dismiss
- Error: 수동 dismiss + traceId copy 버튼

### 3.10 Loading

- 전체 페이지: centered `CircularProgressIndicator` + "불러오는 중…"
- 테이블 갱신: skeleton rows 5개
- 버튼: inline spinner, label 유지
